Oke sebelum saya mulai,pastikan kalian mempunyai:

* Akun GitHub
* Dokumen seperti KTM atau Transkrip Nilai
* Email dengan domain Kampus atau Sekolah

> Contoh: boybdul@binus.ac.id

Kalau sudah yakin,berikut langkah-langkahnya:

* Silahkan kalian buka education.github.com
* Pilih GitHub Student Developer Pack dan pilih Get the Pack
* Selanjutnya kalian akan di arahkan oleh GitHub untuk mengajukan permohonan
* Silahkan kalian isi Form tersebut
* Jika sudah silahkan klik Submit Your Information.

Terakhir jika permohonan atau pengajuan kalian di terima oleh GitHub maka kalian akan mendapatkan email cinta atau sambutan dari Github dengan Judul:

> Welcome to the GitHub Student Developer Pack

FYI nih dari pengalaman saya,kalian akan mendapatkan Jawaban yang entah itu Ditolak atau Diterima oleh GitHub sekitar 1 Mingguan. Kenapa lama sekali? Karena Verifikasi yang dilakukan adalah Manual.

Sekedar tips,saat saya mengajukan pertama kali itu ditolak guys padahal sudah digantung selama 1 Minggu sama GitHub. Saya ditolak karena Dokumen yang saya Upload itu berupa KTM ternyata itu saja tidak cukup bagi GitHub,nah yang kedua saya Upload Transkrip Nilai dan hasilnya diterima.
